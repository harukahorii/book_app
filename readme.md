#  使い方
「投稿するボタン」を押して、読んだ本の「題名」「感想」「それを踏まえて行動すること」「いつまでにやるかの期限」を入力して保存ボタンを押す

#  なんのために作ったか？
本を読んだ時に、読み終わって満足してそこから行動に伴わないという状態を改善し、何を学んでそこからどのように行動するかいつまでにやるかを記録できるようにすることで、学びを行動に繋げられるようにする。そしてそれを繰り返して、自分の夢を叶えられるようにする。
